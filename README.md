## ✨ Pemenuhan Fitur (Sesuai Rubrik)
**🟢 Level 1 (Core)**
- `CREATE`, `READ`, `DELETE` berfungsi penuh dengan logika *array* yang tepat.
- Persistensi data menggunakan `@react-native-async-storage/async-storage`.
- UI menggunakan `FlatList`, memiliki `keyExtractor`, dan menampilkan `ListEmptyComponent` saat kosong.

**🟡 Level 2 (Pengembangan)**
- **UPDATE:** Menandai tugas selesai (teks dicoret).
- **Dark Mode Tersimpan:** Tema *Midnight Blue* elegan yang preferensinya disimpan di memori.
- **Konfirmasi Hapus:** Mencegah penghapusan tidak sengaja.

**🔴 Level 3 (Bonus)**
- **Timestamp:** Tanggal dan waktu pembuatan otomatis tersimpan dan ditampilkan pada setiap *item*.
